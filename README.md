# Dap IN — Marketing Website

Real plans. Real people. Real life.

Four-page static marketing site for Dap IN, built with plain HTML/CSS/JS — no frameworks, no build step.

## Pages

- `index.html` — Home (animated hero, how it works, FAQ)
- `about.html` — About Us
- `host-guide.html` — Host Guide
- `contact.html` — Contact Us

## Local preview

Open `index.html` in any browser, or run:

```
python3 -m http.server 8000
```

## Deployment

Hosted with GitHub Pages: Settings → Pages → Deploy from a branch → `main` / root.
